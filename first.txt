Gitの練習をしていきます。
基本的なワークフローを学びます。
git add file name
git commit
git commit -v
git rm
git status 現在の状況の確認
git log →jでスクロールダウン、kでスクロールアップ、qで終了
git log --oneline
git log -n 3
git diff ステージとの差分
git diff HEAD　ステージとコミットの差分
git remote add origin Address →ローカルリポジトリをリモートリポジトリに登録できる。以後はoriginという名前でGitHubリポジトリにアクセス可能
git push origin master　→OriginというリモートレポジトリにMasterブランチを送信するという意味
git push origin feature
